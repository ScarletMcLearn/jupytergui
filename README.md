# Simpli

---
## Install Simpli

### For Mac OS X
```bash
# Install brew
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

# Install git
brew install git

# Download simpli
git clone https://github.com/UCSD-CCAL/simpli.git

# Install simpli
cd simpli
pip install .
```


### For Linux
```bash
# Install git
sudo apt-get install git

# Download simpli
git clone https://github.com/UCSD-CCAL/simpli.git

# Install simpli
cd simpli
pip install .
```
